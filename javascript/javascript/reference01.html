자바스크립트 선언문

  • 자바스크립트 구문을 실행하기 위해서는 선언문을 입력해야 합니다
  • head 영역이나 body 영역에 선언하면 됩니다.

자바스크립트 주석처리

  • 자바스크립트에서는 방법이 두 가지가 있습니다. //를 넣으면 // 뒤부터 그 줄의 끝까지가 주석이 됩니다.
  • 여러 줄을 주석 처리하고 싶다면 /*와 */로 감쌉니다.

코드 입력시 주의사항

  • 1. 자바스크립트는 대소문자를 구분하여 작성한다.
    EX) 날짜 객체 생성: New date(); (X)
    EX) 날짜 객체 생성: New Date(); (O)
  • 2. 코드 한 줄을 작성한 후에는 세미콜론(;)을 쓰는 것이 좋다.
    (세미콜론을 쓰지 않으면 한 줄에 2개의 코드를 작성할 경우 오류가 발생한다.)
  • EX) document.wrute("hi") document.write("bye") (X)
  • EX) document.wrute("hi"); document.write("bye"); (O)
  • 3. 한 줄에 한 문장만 작성하는 것이 가독성을 위해 좋다.
  • 4. 문자형 데이터를 작성할 때는 큰따옴표(" ")와 작은따옴표(' ')의 겹침 오류를 주의해야 한다.
  • EX) document.wrute('외부 글자 안의 "내부 글자"가 있다.'); (O)
  • 5. 코드를 작성할 때 중괄호{} 또는 소괄호()의 짝이 맞아야 한다.
  • EX) document.wrute("hi"; (X)
  • EX) document.write("bye"); (O)>
